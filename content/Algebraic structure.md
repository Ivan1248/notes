Let $n\in \N$.

An **algebraic structure with $n$ operations** is an $(n+1)$-[[Tuple|tuple]] $(S, \circ_1, .., \circ_n)$ such that:
- $S$ is a [[Set|set]],
- for each $i\in \{1, 2, \ldots, n\}$, $\circ_i$ is a [[Binary operation|binary operation]] from $S\times S\to T_i$ where $T_i$ is a [[Set|set]].

## Notes

> [!NOTE] The $n=1$ special case  
> An **algebraic structure with 1 operation** is an [[Ordered pair|ordered pair]] $(S, \circ)$ such that:
> - $S$ is a [[Set|set]],
> - $\circ$ is a [[Binary operation|binary operation]] from $S\times S\to T$, where $T$ is a [[Set|set]].
