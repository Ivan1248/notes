A **probability function** is a [[Function|function]] $P\colon X\to[0\bidot 1]$ where $X$ is a [[Countable set|countable set]] and $\sum_{x\in X} P(x) = 1$.  

## Probability function with an uncountable support

A **probability function** is a [[Function|function]] $P\colon X \to [0\bidot1]$ where, if $X$ is a [[Uncountable set|countable set]],
$$
\sum_{x\in X} P(x) = 1 \text,
$$
and, if $X$ is an [[Uncountable set|uncountable set]],
$$
\sum_{x\in X} P(x) \leq 1 \text.
$$
## Probability function 

A **probability function** is a [[Function|function]] $P\colon X\to[0\bidot 1]$ such that there exists a [[Random variable|random variable]] $\rvar x\colon\Omega\to X$ such that $P$ is the [[Probability function of a random variable|probability function of]] $\rvar x\colon\Omega\to X$.
