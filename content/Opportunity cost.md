Let $A$ be the option that we choose. Let option $B$ be the best alternative option. The **opportunity cost** of choosing option $A$ is the value of choosing option $B$.

## Example

Let there be 2 options: $A$ and $B$.

Option $A$ has:
- price $P_A = 30$,
- gross value $V_A=50$,
- value $U_A = V_A - P_A = 20$.

Option $B$ has:
- price $P_B = 10$,
- gross value $V_B$,
- value $U_B=V_B-P_B$.

_How much would you have to value option $B$ to be indifferent to the choice?_

Option $B$ should have:
- value $U_B = U_A = 20$,
- gross value $V_B= U_B + P_B = 20 + 10 = 30$.

In this case, the opportunity cost of choosing $A$ is $U_B=20$.
