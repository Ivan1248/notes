A **distance function** on a [[Set|set]] $A$ is a [[Function|function]] $d\colon A\times A \to \R$ such that
- $\forall x,y\in A\colon (d(x, y) > 0 \iff x \neq y) \wedge (d(x, y) = 0 \iff x = y)$ (positivity or $0$ only when $x=y$),
- $\forall x,y\in A\colon d(x, y) = d(y, x)$ (symmetry),
- $\forall x,y,z\in A\colon d(x, y) + d(y, z) \geq d(x, z) \equiv x = y$ (triangle inequality).
