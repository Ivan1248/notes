**Deontology** is an [[Action-based ethical framework|action-based ethical framework]] that assigns [[Ethical value|ethical value]] to [[History of an agent|histories of agents]] $H\subset P\times A$ according to a finite **set of rules**

The set of rules can be formalized as a function $R\colon P\times A\to V$, where $V$ is a finite set of [[Ethical value|ethical values]].

## Notes

- A deontology assigns an ethical value from $V$ to each perception-action pair from $P\times A$.
