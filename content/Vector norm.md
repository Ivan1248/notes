## Real vector space

A **norm** on a [[Real vector space|real vector space]] $V$ is a [[Function|function]] $\lVert\cdot\rVert\colon V\to\R_{\geq 0}$ such that:
- $\forall \vec x \in V : \lVert \vec x \rVert = 0 \iff \vec x = \cvec 0$ ([[Positive definiteness|positive definiteness]]),
- $\forall \vec x \in V, \lambda \in \R : \lVert \lambda \vec x \rVert = \lvert\lambda\rvert \lVert \vec x\rVert$ ([[Positive homogeneity|positive homogeneity]]),
- $\forall \vec x, \vec y \in V : \lVert \vec x +\vec y \rVert \leq \lVert\vec x\rVert + \lVert \vec y\rVert$ ([[Triangle inequality|triangle inequality]]).
