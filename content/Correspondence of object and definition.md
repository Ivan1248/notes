## 1

We are looking for a way to say whether an object corresponds to a definition $D$.

## 2

Let $x$ be a description of an object.  
Assume that $x$ corresponds to a definition $d$ iff there is a function $m$ that maps $x$ to $d$.  
Such a function $m$ always exists: for example $m = \{(x, d)\}$.  
Therefore every object corresponds to definition $d$.
